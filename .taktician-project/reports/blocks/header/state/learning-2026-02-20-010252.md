# Learning Analysis — Revision 3

## Mistake

Mobile menu was not displaying properly after the revision 16 structural restructure. The navigation was moved inside the right column to fix desktop logo spanning, but this caused the mobile menu to become inaccessible or malfunction on smaller screens.

## Expected

Mobile menu should be fully functional and properly positioned across all breakpoints, matching the desktop layout restructure without regression.

## Root Cause

The right column was constrained inline on mobile instead of wrapping to full width. Moving `nav` into the right column worked correctly for desktop (allowing the logo to span the full visual height of two rows), but the parent flex container didn't explicitly handle wrapping on smaller breakpoints. This caused both the left and right columns to remain inline and squeezed, making the right column (and the mobile menu inside it) compressed and inaccessible.

The fix used responsive utility classes: `max-md:flex-wrap` on the parent container and `max-md:w-full max-md:order-last` on the right column, ensuring it wraps to a full-width second row on mobile while preserving the desktop inline layout.

- **Type:** design-interpretation
- **Responsible Phase:** development

## Suggestions

### Project-Specific (applied to taktician.md)

1. **[ADD] Layout Patterns**: Responsive flex-wrap for structural reflows: When restructuring flex layouts for desktop optimization (e.g., moving nav into columns for logo spanning), always test mobile wrapping behavior. Use `max-md:flex-wrap` on the flex parent and `max-md:w-full max-md:order-last` on child columns that should reflow to full-width rows on mobile. Desktop `gap` and `flex` classes remain inert on mobile when `max-md:` prefixes are used.
   - *Rationale:* Desktop-optimized structural changes can break responsive layouts if flex wrapping isn't explicitly controlled across breakpoints. This pattern ensures mobile menu accessibility is preserved when restructuring for desktop logo spanning or other desktop layout requirements.
