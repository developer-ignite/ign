Create the Tabs block. Use the tabs block type instructions.

Desktop design: https://www.figma.com/design/bpRK9A1l2nTbWfMg1pX5yK/IGN-%7C-WEBSITE--INTERNAL-?node-id=785-34910&m=dev and https://www.figma.com/design/bpRK9A1l2nTbWfMg1pX5yK/IGN-%7C-WEBSITE--INTERNAL-?node-id=807-58405&m=dev
Mobile design: https://www.figma.com/design/bpRK9A1l2nTbWfMg1pX5yK/IGN-%7C-WEBSITE--INTERNAL-?node-id=785-33971&m=dev

The background of the items will depend on the page accent color. You should use /var/www/ign/wp-content/themes/ign/blocks/Accordion as reference for the application of the accent color. One tab will always have to be selected on desktop.

On mobile it's going to work like accordions. On mobile the user can have multiple or no tabs (accordions) open. When changing  from mobile to desktop, if all tabs (accordions) are closed, the first one should be selected.
