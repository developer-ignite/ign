Create a block named Text. It will have two columns, the first will have the theme heading, the second will accept inner blocks. Allow text blocks (paragraph, list...). On the inner blocks div add the class "discourse". There is no need for a reverse columns option.

Use the /var/www/ign/wp-content/themes/ign/.taktician/instructions/block-types/content-with-media.md instructions as a start point.
